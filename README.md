# my-c-learning: My C Language Learning Diary

**This repository contains all the code exercises I've written while self-learning C programming. Each file is named by date, creating a timeline of my learning journey.**

---

## 📅 About This Repository

This is my **"Learning Diary"** - a collection of approximately 100+ code files organized by date. It documents my complete journey from writing my first "Hello World" to implementing data structures and algorithms.

### My Learning Path

*   **The Beginning:** First steps with `printf()`, variables, loops (`for`, `while`), and conditional statements.
*   **Going Deeper:** Nested loops, arrays, and string manipulation exercises.
*   **Challenging Concepts:** Mastering pointers, dynamic memory allocation (`malloc`, `free`), and memory management.
*   **Algorithms Practice:** Implementing classic algorithms like Binary Search, Bubble Sort, Selection Sort, and prime number finding.
*   **Advanced Topics:** Working with structures, arrays of structures, and building custom string functions beyond `<string.h>`.

## 💡 My Learning Philosophy

I believe **"Code is the best note"**. Even the simplest, most naive code files are kept here because they represent genuine learning moments. There's a special joy in debugging a complex nested loop or successfully managing memory without leaks.

## 🚀 From Practice to Projects
